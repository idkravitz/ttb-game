Installation
------------
The main requirements for successfull installation are: _>=python 3.1.2_ and _>=sqlalchemy 0.6_

Python can be installed by simply downloading installer for your OS here http://www.python.org/download/
or if you are on a linux-based OS by package manager. For instace Ubuntu users can do it by:
    $ sudo aptitude install python3

For a quick installation of sqlalchemy it's better to pip utility or easy\_install, if you have so, just use one 
of the following
    $ sudo easy_install SQLAlchemy
    $ sudo pip install SQLAlchemy

Or if you can't or don't want to have none of this nice tools you can download SQLAlchemy directly here http://www.sqlalchemy.org/download.html
and execute
    $ sudo python setup.py install
